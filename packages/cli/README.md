# harless

Minimal, agent-agnostic AI Coding Harness — one `npx harless init` to add
skills and scripts under `.agents/`, merge MCP config, and extend `AGENTS.md`.

## Quick start

```bash
npx harless init        # interactive
npx harless init --yes  # defaults
```

## Commands

```bash
npx harless add <module> | remove <module> | update | doctor
```

Full docs, design notes, and contributing: [github.com/nicepkg/harless](https://github.com/nicepkg/harless).
