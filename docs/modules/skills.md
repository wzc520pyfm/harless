# skills — Discipline Bundle

5 always-on discipline skills that improve agent behavior across all tasks.

| Skill | Trigger |
|---|---|
| brainstorming | Before any creative work |
| tdd | Before writing implementation code |
| systematic-debugging | When encountering a bug |
| verification-before-completion | Before claiming work is done |
| receiving-review | When receiving code review feedback |

## Files

- `.agents/skills/brainstorming/SKILL.md`
- `.agents/skills/tdd/SKILL.md`
- `.agents/skills/systematic-debugging/SKILL.md`
- `.agents/skills/verification-before-completion/SKILL.md`
- `.agents/skills/receiving-review/SKILL.md`

## Usage

These are "always-on" — the AGENTS.md index tells agents to read them
whenever the matching trigger condition is met. No script required.
