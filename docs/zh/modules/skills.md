# skills — 纪律技能包

5 个始终生效的纪律 skill，提升 Agent 在所有任务中的表现。

| Skill | 触发条件 |
|---|---|
| brainstorming | 任何创造性工作之前 |
| tdd | 编写实现代码之前 |
| systematic-debugging | 遇到 bug 时 |
| verification-before-completion | 声称工作完成之前 |
| receiving-review | 收到代码审查反馈时 |

## 文件

- `.agents/skills/brainstorming/SKILL.md`
- `.agents/skills/tdd/SKILL.md`
- `.agents/skills/systematic-debugging/SKILL.md`
- `.agents/skills/verification-before-completion/SKILL.md`
- `.agents/skills/receiving-review/SKILL.md`

## 用法

这些是"始终生效"的 —— AGENTS.md 索引告诉 Agent 在匹配的触发条件出现时
读取对应文件，无需脚本。
